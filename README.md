# 🏢 HR Intelligent RAG Chatbot (Gemini Version)

ระบบ RAG Chatbot สำหรับฝ่ายบุคคล (HR) ค้นหาคำตอบจากเอกสารนโยบายบริษัทและตอบคำถามพนักงานโดยใช้ **Gemini API** รันภายในโปรเจกต์เดียวกันแบบโลคอล 100%

---

## 🚀 โครงสร้างโปรเจกต์
- `documents/` : โฟลเดอร์เก็บเอกสาร PDF นโยบายต่างๆ
- `generate_docs.py` : สคริปต์จำลองสร้างเอกสาร PDF ภาษาไทยสำหรับทดสอบ
- `ingest.py` : สคริปต์สแกนเอกสาร PDF, แบ่งเป็น chunks, แปลงเป็นเวกเตอร์ด้วย Gemini Embedding และบันทึกลงฐานข้อมูล ChromaDB
- `app.py` : เว็บแอปพลิเคชันหน้าจอแชทสวยงาม พัฒนาด้วย Streamlit
- `requirements.txt` : รายการไลบรารีที่ต้องติดตั้ง
- `.env` : ไฟล์จัดเก็บ `GEMINI_API_KEY` (ห้ามแชร์คีย์นี้)

---

## 🛠️ ขั้นตอนการติดตั้งและการรันระบบ

### 1. การติดตั้ง Python & ไลบรารี
เปิด Terminal / Command Prompt ในโฟลเดอร์ของโปรเจกต์นี้ จากนั้นติดตั้งไลบรารีด้วยคำสั่ง:
```bash
pip install -r requirements.txt
```

### 2. ตั้งค่า API Key
1. คัดลอกไฟล์ `.env.example` และเปลี่ยนชื่อเป็น `.env`
2. ไปที่ [Google AI Studio](https://aistudio.google.com/) เพื่อสมัครรับ **Gemini API Key** ฟรี
3. เปิดไฟล์ `.env` และกรอกคีย์ของคุณดังนี้:
   ```env
   GEMINI_API_KEY=AIzaSy...ของคุณ...
   ```

### 3. สร้างเอกสาร PDF ตัวอย่าง
รันคำสั่งด้านล่างเพื่อสร้างไฟล์ PDF ภาษาไทย 3 ไฟล์ ได้แก่ กฎการลาพักร้อน, เบอร์ติดต่อภายใน และ นโยบายการลาป่วย/ลากิจ ในโฟลเดอร์ `/documents`:
```bash
python generate_docs.py
```

### 4. นำเข้าข้อมูลเอกสารเข้าสู่ Vector Database
รันสคริปต์ประมวลผลคำเพื่อสร้างฐานข้อมูลเวกเตอร์ ChromaDB:
```bash
python ingest.py
```
*ระบบจะสร้างโฟลเดอร์ `chroma_db/` ขึ้นมาสำหรับใช้ค้นหาข้อมูล*

### 5. รันแอปพลิเคชันเว็บหน้าแชท
เปิดหน้าจอใช้งานผ่านเบราว์เซอร์ด้วยคำสั่ง:
```bash
streamlit run app.py
```
*เบราว์เซอร์จะเปิดหน้าระบบแชทให้อัตโนมัติที่ `http://localhost:8501`*

## 📝 คำถามที่ใช้ทดสอบความถูกต้อง (RAG Test Cases)
คุณสามารถพิมพ์ถามตอบคำถามเหล่านี้บนหน้าจอแชทเพื่อตรวจสอบความถูกต้องของระบบ:
1. **ถามเรื่องการลาพักร้อน:** *"ลาพักร้อนได้กี่วันต่อปี"* 
   * (ควรตอบ: 10 วันต่อปี สำหรับอายุงาน 1-3 ปี และเพิ่มตามอายุงาน พร้อมอ้างอิงจาก `vacation_policy.pdf`)
2. **ถามเบอร์ภายใน:** *"เบอร์ติดต่อ IT คืออะไร"* 
   * (ควรตอบ: 1101 อีเมล it.support@company.com พร้อมอ้างอิงจาก `internal_contacts.pdf`)
3. **ถามเรื่องการลาป่วย:** *"ลาป่วยต้องแจ้งล่วงหน้ากี่วัน/แจ้งอย่างไร"* 
   * (ควรตอบ: ต้องแจ้งหัวหน้างานโดยเร็วที่สุดอย่างน้อย 30 นาทีก่อนเริ่มงาน หรือภายใน 09:00 น. ของวันที่เริ่มหยุด พร้อมอ้างอิงจาก `general_policy.pdf`)
4. **ถามคำถามภายนอกนอกเหนือเอกสาร:** *"สัปดาห์หน้าฝนจะตกไหม"* หรือ *"ขอสูตรทำส้มตำหน่อย"*
   * (ควรตอบ: ไม่พบข้อมูลดังกล่าวในเอกสารนโยบายบริษัท และแนะนำให้ติดต่อฝ่าย HR ตามเงื่อนไขของ System Prompt)
